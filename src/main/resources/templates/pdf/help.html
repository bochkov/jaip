Для того чтобы разбить pdf-файл

  1. Идем по ссылке разбить
  2. Выбираем файл
  3. Указываем номера страниц, которые должны остаться. Можно указывать как отдельные страницы через запятую, так и диапазон. Например:
    • 3,8,15 - новый файл будет содержать 3-ю, 8-ю и 15-ю страницы исходного файла
    • 3-9 - новый файл будет содержать все страницы с 3-ей по 9-ю включительно
    • -5 - новый файл будет содержать первые пять страниц исходного файла
    • 4- - новый файл будет содержать с 4-й по последнюю страницу исходного файла
    • указанные способы можно комбинировать между собой, например -5,8,15-24
  4. По умолчанию, их указанных страниц собирается новый pdf-файл. Также указанные страницы можно получить отдельно друг от друга. Для этого существует кпопка «В один файл»
  5. Нажимаем на кнопку Разбить!
  6. PROFIT

Для того чтобы собрать pdf-файл

  1. Идем по ссылке собрать
  2. Выбираем файлы, которые необходимо объединить в один. Внимание! Порядок страниц в новом файле определяется исходя из имен исходный файлов. Первыми будут добавлены файлы начинающиеся с цифр, потом - начинающиеся с букв латинского алфавита, затем - букв русского алфавита.
  3. Нажимаем на кнопку Собрать!
  4. PROFIT
